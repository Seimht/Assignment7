# Jokebook App – Assignment 7

**Author:** Seim Habte  
**Course:** CSC 372 – Web Development  
**Date:** November 2nd 2025  

This project is a simple Express.js + PostgreSQL (Neon) web app that lets users browse, view, and add jokes.  
It demonstrates RESTful API design, async database handling, and client-side interaction using JavaScript.

## Features

- Built with Node.js, Express, and PostgreSQL (via Neon)**  
- Four API endpoints:  
  - `GET /jokebook/categories` – lists all joke categories  
  - `GET /jokebook/category/:category` – shows jokes by category  
  - `GET /jokebook/random` – returns a random joke  
  - `POST /jokebook/joke/add` – adds a new joke  
- Fully functional front end (HTML, CSS, JS)  
- Environment variables managed through `.env`
